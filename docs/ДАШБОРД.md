# Дашборд ГородОК

Веб-интерфейс на Streamlit. Точка входа: `dashboard/app.py`, запуск через `run.ps1`.

## Боковая панель

- **Навигация** — Главная, Графики, Отчёты, Сравнение.
- **История отчётов** — список папок `Отчет от …` из `storage/output`. Выбор отчёта обновляет данные на всех страницах.
- **Загрузка Excel** — файл `.xlsx` для новой обработки.
- **Режим обработки** — пресет пайплайна (см. [РЕЖИМЫ_ОБРАБОТКИ.md](РЕЖИМЫ_ОБРАБОТКИ.md)).
- **Дополнительные параметры** — лимит строк, случайная выборка, размер батча LLM, число кластеров.
- **Запустить обработку** — запуск `analyze/pipeline.py` (или только очистки).

После успешного анализа активный отчёт копируется в `dashboard/data/reports/`.

## Главная

- Кнопки скачивания: полный Excel, топ районов, PDF-справка, ZIP.
- KPI: всего обращений, выявлено проблем, районов, средняя тяжесть, главная тема.
- Блок статистики очистки (сколько отфильтровано и почему).
- Распределение по степени тяжести (высокая / средняя / низкая).
- **ТОП-3 проблемных района** — карточки с ключевыми проблемами.
- Таблица обращений с сортировкой.

## Графики

- Селектор **«Выберите муниципалитет для подсветки»** — золотым выделяется выбранный муниципалитет на карте и графиках (данные не фильтруются).
- Сетка 2×4 (пары блоков в ряд):
  1. Карта Омской области (хороплет по числу проблем) ↔ ТОП-10 муниципалитетов по тяжести
  2. ТОП-10 муниципалитетов ↔ ТОП-10 районов
  3. Распределение обращений по муниципалитетам ↔ круговая диаграмма проблем
  4. Классификация обращений ↔ динамика по датам (подписи дат на русском)

## Отчёты

- Те же KPI и статистика очистки.
- Текстовая **краткая сводка** из файла `… — сводка.txt`.
- Таблица обращений.

PDF-справка и файлы для скачивания — на **Главной**.

## Сравнение

- Выбор двух отчётов из истории.
- Карточки «Первый» / «Второй» отчёт.
- Сводные метрики с дельтами и процентами.
- Таблица изменений по районам.
- Блок **«Кратко»** — рост и снижение числа проблем.
- Кнопка **«Скачать сравнение (PDF)»**.

Требуется файл `… — топ районов.xlsx` (листы Обзор и Все районы) в каждом из сравниваемых отчётов.
