# ГородОК

Система анализа обращений граждан из Excel: очистка данных, определение проблем и серьёзности, отчёт Top-3 / Top-10 по районам, веб-дашборд.

## Установка

```powershell
.\setup.ps1
.\.venv\Scripts\python.exe tools\download_model.py
```

Тестовый Excel положите в `storage\samples\`.

## Запуск

**Дашборд:**

```powershell
.\run.ps1
```

Загрузите `.xlsx` в боковой панели, выберите режим обработки и нажмите **Запустить обработку**.

**Командная строка (рекомендуемый режим):**

```powershell
.\.venv\Scripts\python.exe analyze\pipeline.py `
  --input "storage\samples\файл.xlsx" `
  --work-dir . `
  --mode cluster --turbo
```

Результаты: `storage\output\Отчет от …\` (`cleaned.xlsx`, `result.xlsx`, `report.xlsx`, `report.pdf`).

## Режимы обработки

| Режим | Назначение |
|-------|------------|
| Только очистка | Фильтрация и нормализация текста |
| Быстрый | Правила, без GPU |
| Гибридный | Правила + LLM для топ-районов |
| Кластерный + turbo | Основной режим для больших файлов |
| Кластерный полный | Больше вызовов LLM |
| Полный LLM | Анализ каждой строки |

Параметры CLI и примеры: [docs/РЕЖИМЫ_ОБРАБОТКИ.md](docs/РЕЖИМЫ_ОБРАБОТКИ.md).
