## Анализ обращений граждан

В исходном файле — десятки или сотни тысяч обращений людей: жалобы на снег, воду, дороги, отопление и т.д. Вручную всё это не прочитать.
Наша программа помогает:
- понять, в каком обращении есть реальная проблема, а где просто благодарность;
- оценить, насколько ситуация серьёзная (от «мелочь» до «критично»);
- кратко описать суть каждой жалобы;
- собрать список самых проблемных районов (топ-3 и топ-10) для руководства.

## Принцип работы
На входе:
- файл Excel с колонками: текст обращения, тема, регион, муниципалитет, населённый пункт и др.

На выходе (в отдельной папке с датой и временем):
-	result.xlsx — каждая строка с пометками: проблема да/нет, тяжесть, краткое описание;
-	report.xlsx — таблицы «Топ-3» и «Топ-10» районов;
-	report.txt — короткая текстовая справка, которую можно показать руководству.


## Стек технологий
- Python — язык, на котором написаны все скрипты.
-	Excel (pandas) — чтение и запись таблиц.
-	Правила и ключевые слова — быстрые проверки без нейросети.
-	Группировка похожих текстов — математический способ найти «похожие» жалобы.
-	Qwen 7B — нейросеть, которая читает русский текст; работает на вашем компьютере, без интернета.
