Папки с результатами анализа
============================

Каждый запуск пайплайна создаёт каталог вида:
  Отчет от дд.мм.гг - чч.мм\

Типичное содержимое:
  … — очищенные данные.xlsx   — после фильтрации
  … — полный анализ.xlsx      — все строки с разметкой
  … — топ районов.xlsx        — Топ-10, Топ-3, все районы, обзор
  … — справка.pdf             — PDF для руководства
  … — сводка.txt              — краткий текст
  cleaning_stats.json         — статистика очистки

Папки не попадают в Git. Список отчётов доступен в дашборде
(боковая панель → история отчётов).

Старые отчёты с именами result.xlsx / report.xlsx тоже открываются.
